Indonesia Jamu-Herbs (IJAH) (known as Ijahv1)
Indonesian Jamu-Herbs Information System (SI-IJAH) is an expert system for Jamu formula prediction.
SI-IJAH utilizes several machine learning approaches, namely: PLS-DA, N-PLS, SVM, and VFI5.
